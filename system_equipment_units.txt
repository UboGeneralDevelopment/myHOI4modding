
装備とユニット解説



ユニットの基本的なステータスは装備とユニットの種類によって決まる。

装備によって決まるステータス
build_cost_ic = <float> #生産コスト
lend_lease_cost = <float> #設計図レンドリースコスト
reliability = <float> #信頼性
maximum_speed = <float> #最高速度
#以下の値は装備充足率がダイレクトにユニットステータスに影響する
defense = <float> #防御値
breakthrough = <float> #突破値
hardness = <float> #装甲化率
armor_value = <float> #装甲値
soft_attack = <float> #対人攻撃
hard_attack = <float> #対装甲攻撃
ap_attack = <float> #貫通
air_attack = <float> #対空

ユニットによって決まるステータス。必要装備が同じでも、ユニット種類によって変化するステータス。
manpower = <int> #必要人的資源
max_strength = <int> #最大耐久値
training_time = <int> #訓練時間
weight = <float> #重量
default_morale = <float> #基本組織率回復率
max_organisation = <int> #最大組織率
combat_width = <int> #戦闘幅
supply_consumption = <float> #補給消費

ユニット、技術、アイデアによって与えることができるモディファー


ユニットや装備の分類はいろいろある。それぞれ以下のようにわけられる。カテゴリー以外の分類はmodでは増やせない。

タイプ
ユニットおよび装備の分類。
複数のタイプに所属可能。ユニットのタイプと、装備のタイプは基本的には一致するが何とも言えない。
自動車歩兵などは歩兵タイプではないが歩兵装備を使うのでよくわからん。
使い道としては、今のところアイデアによる装備ボーナスは装備のタイプが参照されることは判明している。
ユニットに設定されたタイプが何か効果あるかといわれると、よくわからん。
ユニットの装備補正は、装備している装備に対してタイプにかかわらず適用されるので、ユニットタイプの意味がよくわからない。

グループ
単一のグループにしか所属不可能。師団編成画面でのグループ選択に用いられ、種類は支援、歩兵、機動、装甲のみ

カテゴリー
ユニットの分類。技術による補正を与える際に用いる。複数カテゴリーに所属可能、技術補正はorで適用される。modで増やせる。
基本的に技術補正は装備ではなくてユニットに乗る形となっている。
