共通フォーカス
基本の国家方針に追加する形で機能する国家方針。意外にシンプル。
共通フォーカスは基本の国家方針中に指定されると、つながっているツリーが芋づる式に導入されるようになっている。
共通フォーカスの最初のものの座標は、国家方針ウィンドウの絶対座標で指定される。
つながってないツリーを同時に導入することはできず、ツリーの最初の方針は基本国家方針の中に別々に明記する必要がある。
しかし、相対位置は共通方針どうしの関係で指定することができる。
中国を基にした定義は以下の通り。効果などの書き方は国家方針と変わらず、方針ツリーというよりも、追加可能な方針として機能している。

shared_focus = {
	id = CHI_invite_foreign_investors 
	icon = GFX_focus_generic_concessions
	x = 40
	y = 0
	offset = { #ここだけ特殊、トリガーに応じて位置を絶対座標に対して相対的に変更することができる。
		x = -6
		y = 0
		trigger = {
			tag = PRC
		}
	}
	}
	cost = 10
	ai_will_do = {
		factor = 1
		modifier = {
			factor = 4
			tag = CHI
		}
	}
	available = {
	}
	bypass = {	
	}
	cancel_if_invalid = yes
	continue_if_invalid = no
	available_if_capitulated = no
	complete_tooltip = {
	}
	completion_reward = {
		add_offsite_building = { type = industrial_complex level = 2 }
	}
}

継続フォーカス
あまりいじる気がないので詳しくはやらない。見てわかるのは、解除条件となる国家方針の達成数はいじれないことである。
こちらは日々のコストを変更可能。
continuous_focus_palette = {
	id = generic_focus
	country = {
		factor = 1
	}
	default = yes
	reset_on_civilwar = no
	position = { x = 50 y = 1000 }
	focus = {
		id = continuous_naval_production
		icon = GFX_goal_continuous_naval_production
		available = {
		}
		idea = continuous_naval_production
		ai_will_do = {
			factor = 1
		}
		supports_ai_strategy = ai_focus_naval
		daily_cost = 1
		available_if_capitulated = yes
	}
}
